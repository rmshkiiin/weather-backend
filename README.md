Weather API Backend (FastAPI)

Weather API Backend

Учебный backend‑сервис на FastAPI для получения данных о погоде через внешний API.

---

📌 Описание проекта

Weather API Backend — это учебный сервер, который принимает запросы от клиента и возвращает данные о погоде по указанному городу. Проект демонстрирует базовые навыки работы с FastAPI, внешними API, логированием и структурой backend‑приложения.

---

🔧 Стек технологий

• Python
• FastAPI
• Uvicorn
• requests
• OpenWeatherMap API
• Git / GitHub


---

🚀 Функционал

• GET /weather?city=Москва — получение текущей погоды
• Обработка ошибок (неверный город, проблемы API)
• Логирование запросов
• Разделение кода на модули

---

▶️ Запуск проекта

Установка зависимостей:

pip install -r requirements.txt


Запуск сервера:

uvicorn app.main:app --reload
